# SpazaStock

## Project Overview
SpazaStock is a web application designed to streamline inventory management for spaza shops, providing a user-friendly interface for tracking stock, managing orders, and optimizing supply chains.

## Features
- **Inventory Management**: Easily track stock levels and manage inventory.
- **Order Management**: Simplified process for managing customer orders and supplier deliveries.
- **Analytics Dashboard**: Gain insights into sales trends and inventory performance.
- **User Authentication**: Secure user accounts with role-based access.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **Deployment**: AWS
- **Version Control**: GitHub

## Quick Start Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Locx26/SpazaStock.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SpazaStock
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```

For more detailed documentation, refer to the wiki or documentation folder.
